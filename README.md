# cellular-wallpapers
Generate PNG images of famous cellular automata and mathematical systems using Python.

<img src="https://i.imgur.com/nyJU0LY.png" width="600" alt="Wolfram Rule 30">

## Features

* Generate images of the **Mandelbrot Set**
* Simulate **Wolfram's Elementary Cellular Automata** using any rule (0–255)
* Simulate **Langton's Ant**
* Save all outputs as PNG images
* Easily configurable simulation parameters

## Implemented Systems

### Mandelbrot Set

Generates a visualization of the Mandelbrot fractal by computing the escape-time algorithm for each pixel.

### Wolfram Elementary Cellular Automata

Simulates one-dimensional elementary cellular automata as described by Stephen Wolfram.

Popular rules include:

* Rule 30
* Rule 90
* Rule 110
* Rule 184

### Langton's Ant

Simulates Langton's Ant, a two-dimensional cellular automaton that exhibits surprisingly complex emergent behavior.

## Project Goals

This project serves as a collection of classic computational systems that demonstrate how simple mathematical rules can produce intricate and often beautiful patterns.

It is suitable for:

* Learning about cellular automata
* Exploring fractals
* Creating procedural artwork
* Educational demonstrations
* Experimenting with emergent behavior
